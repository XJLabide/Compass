## Description
Build the `/settings` page: unit system, timezone, protein target, weekly gain target, sign-out. Reads/writes `users/{uid}/profile`.

## Approach
- Form uses controlled inputs; saves on blur + visible "Saved" status indicator (debounced).
- Timezone: auto-detect via `Intl.DateTimeFormat().resolvedOptions().timeZone`; let user override via a searchable select of IANA zones.
- Unit toggle is display-only — does NOT rewrite history (per gap analyst).
- Sign-out button at the bottom, separated by a divider.
